# Kalkulator CLI

Projekt grupowy — automatyzacja budowania oprogramowania.
Autorzy: Mykola Kryvolap, Tymofii Khatmullin, Danylo Kokhaniuk, Vladyslav Kovalov, Ostap Avramkin

## Uruchomienie
python main.py

## Testy
python -m unittest testy.py

## Dostepne operacje
- Dodawanie, odejmowanie, mnozenie, dzielenie
- Potega: 2^3
- Pierwiastek: math.sqrt(9)
- Procenty: 50%, 200+10%, 200-10%
- Nawiasy: (2+3)*4


## Przyklady
>>> 2+2
4
>>> 200+10%
220.0
>>> math.sqrt(9)
3.0
