Autorzy: Mykola Kryvolap, Tymofij , Danylo Kokhaniuk, Vladislav, Ostap Avramkin

## Dostepne operacje

- Potega: 2^3
- Pierwiastek: math.sqrt(9)
- Procenty: 50%, 200+10%, 200-10%
- Nawiasy: (2+3)*4
